Clone dotfiles repository where you want.

Run very basic scripts `zsh_installer.sh` & `vim_installer`.

```bash
$ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/vundle
$ vim +BundleInstall

```
